もくてき

iOS, Androidのみで提供するサービスではjQueryなどのライブラリを使わず、ネイティブで実装した方がパフォーマンス良いんじゃないかと思い、以前つくったjQuery.jcflickというライブラリを元に再開発してみた。名前もかっこよくしてみた。MIT Licenseなんだお。

DEMO

iPhone, Android, iPadなどで動きが確認できるお。

セットアップ

タグ内にzflick.cssとzflick.jsを読み込む。

<link rel="stylesheet" href="zflick.css">
<script type="text/javascript" src="zflick.min.js"></script>

下記のようにHTMLを組む。高速化のためにJSで動かす要素はID指定になっている。

<div id="zflickjs">
  <div id="zcontents" class="group contents">
    <div class="zcol"><a href="#" class="zlink"><img src="./images/0.jpg"></a></div>
    <div class="zcol"><a href="#" class="zlink"><img src="./images/1.jpg"></a></div>

<div class="group" style="margin-top: 0.5em;">
  <span id="prev" class="zflickClickBtn" style="float: left;">prev</span>
  <span id="next" class="zflickClickBtn" style="float: right;">next</span>
</div>

new zflickjs({})で初期化

<script>
var init = function(){
  new zflickjs({
    id: 'zflickjs',
    contents: 'zcontents',
    col: 'zcol',
    btn: {
      prev: 'prev',
      next: 'next'
    },
    callback: function(){
      //コールバック関数が使えるお
    }
  });
}
window.addEventListener('load', init, false);
</script>
