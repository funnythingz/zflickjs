zflickjs

iOS, Android に特化した超軽量なflickjsを頑張ってつくるよ！CSS3 + JavaScript で。

もくてき

iOS, Androidのみで提供するサービスではjQueryなどのライブラリを使わず、ネイティブで実装した方がパフォーマンス良いんじゃないかと思ってみた。

セットアップ

タグ内にzflick.cssとzflick.jsを読み込む。 
<link rel="stylesheet" href="zflick.css">
<script type="text/javascript" src="zflick.js"></script>

new zflickjs({})で初期化

<script>
var init = function(){
  new zflickjs({
    id: 'zflickjs',
    contents: 'zcontents',
    col: 'zcol',
    margin: 10,
    btn: {
      prev: 'prev',
      next: 'next'
    }
  });
}
window.addEventListener('load', init, false);
</script>
